We also have data for Q1 2016. Can compare this to the data for Q1 2015, use that year-on-year rise (5.25%) to project figures for the full year 2016. That gives us 840M total, 664M domestic, 176M international.
Now we need to project ahead for 2017 and 2018. Simplest method for doing this is a linear model, as shown on this slide. As you can see, the projection for 2016 is already well above this line, and passenger volume in 2017 and 2018 would increase only slightly.
[SLIDE showing linear model using data 2008-2016)
That sort of projection can be justified by the US data, but does not make sense in the context on this MarketLines data. We will discuss the Rest of World data more in a few minutes, but MarketLines is estimating year-on-year growth of more than 6% in that data. It’s reasonable to conclude then that the same market factors driving growth outside the US will mean at least some growth in the US market. That conclusion is given more weight by the strong figures for Q1 2016.
So instead of a linear model we can see the quadratic line in this chart. It fits the data we have rather better. However it may still be too hopeful: should we expect that US passenger volumes will continue to rise at this rate, reaching 1B by 2019?
[SLIDE showing polynomial model using 2008-2016 data]
So let’s project growth for 2017-2018 at 4%: that is a little more conservative than the Rest of World data but still strong.
Comment on historical data given and estimated volumes, explain findings and implications for the US Airline industry
Data start at 2008, immediately after the financial crisis. 2009 shows a contraction, then growth is relatively small (<3%) until 2015, which shows a 5% increase. 2016 Q1 data also shows 5% year-on-year growth from 2015, so we can expect 2016 to continue that trend.
HOWEVER, we must be careful about these projections. If we extend our dataset backwards from 2008 we can see that passenger volumes increased quite steeply 2003-2007, then fell sharply 2007-2009 before beginning a steady increase. That steady increase is fragile to market shocks such as 9/11 or the financial crisis of 2008.
